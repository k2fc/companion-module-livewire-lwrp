## LWRP

This module will connect to the LWRP server running on an Axia node or other Livewire device via TCP (by default on port 93)

### Actions

- Set GPI state
- Set GPO state
- Set source address for a destination

### Feedbacks

- GPI Pin State - `true` to indicate the pin state is low
- GPO Pin State - `true` to indicate the pin state is low
- Destination Source Address - `true` when the specified source is routed to the specified destination
